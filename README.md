# Todo

A personal weekly/daily todo planner built with Next.js.

## Features

- **Today** and **This week** lists side by side (stacked on mobile, today on top)
- Drag-and-drop between week and today (Trello-style)
- Incomplete day items roll to the next day at midnight (highlighted in orange)
- Incomplete week items roll to next week at the start of the week
- **Future** backlog and **Next week** planning views
- **History** of completed items from previous weeks (read-only)
- Local storage persistence (Google auth coming later)

## Run locally

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000).
